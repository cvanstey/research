Heritage Calendar Maker

Custom photo calendar generator combining family history, multi-country holidays, and bilingual date labeling — built for historical societies and family memory preservation.

PythonPILCustom
View on Hugging Face →
🔒 Request access — not for public download

What It Does

Generates a full 12-month photo calendar as print-ready PDFs (300 DPI), one family photo per month, with bilingual (English/Italian) month and weekday names, and three layered event types rendered directly on the calendar grid: national holidays for multiple countries, family events (birthdays, anniversaries, deaths, immigration dates), and local/regional historical events.

Configurable

  • Year, page size, DPI, font sizes, and color palette via a single config file
  • Holiday countries (any country supported by the holidays Python library)
  • Family and historical events loaded from simple CSV files — no code editing required to update year to year
  • Automatic age/years-since calculations for birthdays, anniversaries, and memorial dates
Built as a personal/family commission — available for custom builds for other families or historical societies. Not distributed as open source or a self-serve template.
