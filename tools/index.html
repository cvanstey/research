Heritage Calendar Maker

Python script that generates print-ready 12-month PDF calendars layering personal photos with public holidays, family milestones, and local historical events — color-coded by source.

PythonPDFPillowArchival

What It Does

Each monthly page places a photo in the top half and a full calendar grid below. The grid pulls from four sources simultaneously: official public holidays for any country (US, IT, JP, MX, and 15+ others), a custom set of fixed cultural dates (Epiphany, Ferragosto, St. Patrick's Day, etc.), a family.csv for birthdays, anniversaries, deaths, and immigration dates with auto-calculated ages and years elapsed, and a history.csv for local or regional historical events.

Color Legend

  • Country holidays — national-flag color per country
  • Family events — green
  • Historical events — brown
  • Weekends — yellow background

Configuration

All settings live in config.json next to the script: year, DPI, page size, font sizes, photo folder path, and which countries to load. Running with setup scaffolds the full folder structure and README. Running normally prompts for year and countries interactively, then outputs one PDF per month.

Stack

  • Pillow — image compositing, grid drawing, PDF output
  • holidays — country public holiday data
  • csv / json — family events and config loading
  • pathlib — cross-platform file handling
Outputs one 8.5×14" PDF per month at 300 DPI. Images named 01.jpg–12.jpg or by month name are auto-discovered from the configured folder.
